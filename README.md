# BookMyShow Clone

A simple front-end clone of the BookMyShow website built using only HTML and CSS.

## Features

- Responsive homepage layout
- Navigation bar
- Movie banners and posters
- Recommended movies section
- Events section
- Clean and modern UI
- Pure HTML and CSS implementation

## Technologies Used

- HTML5
- CSS3

## Project Structure
BookMyShow-Clone/
│
├── index.html
├── style.css
├── images/
└── README.md

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. you can also run by the link in about section

## Learning Outcomes

- HTML page structuring
- CSS styling and layouts
- Flexbox/Grid usage
- Responsive design basics
